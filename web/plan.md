# **Web dev plan**
## Front-end (giao diện tương tác)
* Khung sườn: `index.html`
* Trang trí: `style.css`
* Logic tương tác: `script.js`
* Các file hình ảnh, âm thanh, video... nằm trong `assets/`

`style.css` và `script.js` có thể merge chung vào `index.html`
## Back-end (core xử lý)
- Logic chính: `app.py`
